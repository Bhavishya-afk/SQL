What is MySQL Workbench?
It’s a GUI tool to work with MySQL databases instead of writing everything in command line.

1.Easy to use (GUI-based)
No need to remember complex commands
You can click → create tables, run queries
Good for beginners and faster work

2.Query execution + result visualization
Write SQL and instantly see results in table format
Easier to debug and analyze data

3.Database design (ER diagrams)
You can visually design tables and relationships
Helps in understanding database structure clearly.
